本目录为可推送到 GitHub 的快照（已排除 high-frequency/backup/ 与缓存）。

建议步骤：

1. 在本目录执行：
   git init
   git add -A
   git status

2. 首次提交（请替换为你的用户名/邮箱）：
   git config user.name "Your Name"
   git config user.email "you@example.com"
   git commit -m "Initial commit: high-frequency options experiment mainline"

3. 在 GitHub 新建空仓库后：
   git remote add origin https://github.com/<user>/<repo>.git
   git branch -M main
   git push -u origin main

说明：数据路径见 high-frequency/config.py；运行前需在父目录放置 trade/quote 等数据文件（见 README.md），这些路径已被 .gitignore 忽略。
